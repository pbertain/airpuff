About AirPuff

AirPuff is a modern aviation weather information system that provides real-time METAR data, route planning capabilities, and historical weather analysis. Built with modern web technologies and designed for pilots, dispatchers, and aviation enthusiasts.

Features:
  • Real-time weather updates via WebSocket
  • Interactive route planning
  • Historical weather charts
  • Mobile-responsive design
  • REST API for integration
Data Sources:
  • Fli-Rite API
  • CheckWX API (backup)
  • TimescaleDB for historical data
